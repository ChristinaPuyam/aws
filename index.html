Activity 1 – Static Website on AWS S3

Activity – 1

Deployment of Static Website using AWS S3 Storage

Aim

To create a static web page and deploy it using Amazon Web Services Simple Storage Service (AWS S3).

Steps Followed

  1. Created a static web page using HTML.
  2. Logged in to the AWS Management Console.
  3. Opened the Amazon S3 service.
  4. Created a new S3 bucket with a unique bucket name.
  5. Disabled "Block all public access" for the bucket.
  6. Uploaded the index.html file into the S3 bucket.
  7. Enabled Static Website Hosting from the bucket properties.
  8. Set index.html as the index document.
  9. Added a bucket policy to allow public read access.
  10. Copied the static website endpoint URL.
  11. Opened the URL in a browser to verify deployment.

Result

The static website was successfully created and deployed using AWS S3 storage and accessed through a public URL.

Status

Website is Live and Deployed Successfully.
